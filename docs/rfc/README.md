# Request for Comments (RFC)

RFCs propose and record substantial technical designs before or during implementation.

ADRs (`docs/adr/`) capture **decisions**. RFCs capture **design proposals** with goals, non-goals, and success criteria.

## Status

- Draft
- Accepted
- Superseded
- Rejected
