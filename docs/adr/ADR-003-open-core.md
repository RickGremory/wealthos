# ADR-003

# Adopt an Open Core Strategy

**Status:** Accepted

---

## Context

The long-term vision includes building a sustainable SaaS while encouraging community participation.

---

## Decision

Adopt an Open Core model.

---

## Public Components

- Core platform
- Documentation
- Architecture
- Basic features

---

## Private Components

Future:

- Premium AI
- Advanced analytics
- Enterprise integrations

---

## Consequences

Allows community growth while preserving commercial opportunities.
