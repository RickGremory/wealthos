# ADR-010

# AI-Ready Architecture

**Status:** Accepted

---

## Context

Artificial Intelligence is a core capability, not an afterthought.

---

## Decision

The architecture should expose structured financial context that AI services can consume.

---

## Examples

Future AI capabilities

- Financial coaching
- Tax estimation
- Goal forecasting
- Cash flow prediction
- Spending analysis

---

## Consequences

The architecture must remain modular and event-friendly.
