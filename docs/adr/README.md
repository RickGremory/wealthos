# Architecture Decision Records

Architecture Decision Records (ADR) document important technical and product decisions made throughout the evolution of WealthOS.

Every ADR contains:

- Context
- Decision
- Alternatives Considered
- Consequences
- Status

---

## Status

- Accepted
- Proposed
- Superseded
- Deprecated

---

## Naming

`ADR-NNN-short-title.md` (e.g. `ADR-001-fastapi.md`)
