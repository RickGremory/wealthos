# 2026-07-23 — Project name: WealthOS

**Type:** Product / brand  
**Status:** Accepted

## Decision

The product is named **WealthOS** — the Financial Operating System for Independent Professionals.

## Why

- Signals an *operating system* for financial life, not another expense tracker
- Aligns with manifesto and vision (decision-making platform)
- Distinct from accounting / banking / trading product names

## Not an ADR

Naming is strategic, not a technical constraint. Architecture ADRs remain in `docs/adr/`.
