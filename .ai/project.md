# WealthOS — Project

High-level product context for AI agents and contributors.

## Project Name

WealthOS

## Mission

Help independent professionals convert income into wealth.

## Architecture

Modular Monolith

## Backend

FastAPI

## Frontend

Nuxt

## Database

PostgreSQL

## Principles

- Transactions are the source of truth.
- Goals never own money.
- Everything belongs to an Organization.
- Always use UUID.
- Always use UTC timestamps.

## Repo layout (brief)

- `backend/` — FastAPI modular monolith
- `frontend/` — Nuxt web application
- `infrastructure/` — deployment and environments
- `docs/` — product, engineering, architecture, roadmap, rfc, epics, decisions, adr
- `specs/` — execution SPECs (sprint contracts)
- `planning/` — backlog, roadmap, milestones, releases
- `prompts/`, `.ai/` — AI guidance and working agreement
- `journal/` — working notes
